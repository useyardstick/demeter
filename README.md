Development
-----------

Create a virtual environment (recommended):

```
python -m venv .venv
source .venv/bin/activate
```

Install development dependencies:

```
pip install -r requirements.dev.txt
```

Install pre-commit hooks:

```
pre-commit install
```

Run tests:

```
pytest
```
